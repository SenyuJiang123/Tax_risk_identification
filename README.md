Step 3: 依据指标得分计算公式及权重，计算企业税务风险总得分，制定企业税务风险等级  
&emsp;&emsp;&emsp;&nbsp;对应代码文件：model.ipynb  

Step 4: 运用模型对行业案例企业做风险评估与分析  
&emsp;&emsp;&emsp;&nbsp;对应代码文件：case.ipynb
